# Installation

LABS requires **Node >= 18** and **npm** (or yarn / pnpm).

## Clone & Install

```bash
git clone <your-repo> my-docs
cd my-docs
npm install
```

## Development

```bash
npm run dev
```

Opens the Vite dev server at `http://localhost:5173`.

## Production Build

```bash
npm run build
npm run preview
```

Builds to `dist/` and serves it locally for preview.
